# DataViz_Assessment1

### Guidelines

1. Check for updates.

```bash
git checkout main
git pull origin main
```

2. Create a new branch.

```bash
git checkout -b <your-branch-name>
```

3. Make your changes.

4. Push your branch to Github.

```bash
git push origin <your-branch-name>
```

5. Create a Pull Request

6. Request a review.

7. Address feedback if there is any.

8. Merge the PR.
